# Easy-Epidemiology Web 테스트 실행 체크리스트

## 🚀 사전 준비 체크리스트

### 개발 환경 설정
- [ ] Node.js 16+ 설치 확인
- [ ] npm 또는 yarn 설치 확인
- [ ] 프로젝트 디렉토리에서 `npm install` 실행
- [ ] 개발 서버 실행 (`npm run serve`)
- [ ] 브라우저에서 `http://localhost:8080` 접속 확인

### 테스트 도구 준비
- [ ] Chrome DevTools 설치
- [ ] Firefox Developer Tools 설치
- [ ] Edge DevTools 설치
- [ ] 성능 모니터링 도구 (Chrome Task Manager)
- [ ] 메모리 사용량 측정 도구

### 테스트 데이터 준비
- [ ] 기본 테스트 데이터 (100명) 준비
- [ ] 대용량 테스트 데이터 (1000명) 준비
- [ ] 엣지 케이스 데이터 준비
- [ ] 정상 Excel 파일 (.xlsx, .xls) 준비
- [ ] 잘못된 형식 파일 준비
- [ ] 대용량 파일 (>10MB) 준비

### 브라우저 준비
- [ ] Chrome 최신 버전 설치
- [ ] Firefox 최신 버전 설치
- [ ] Edge 최신 버전 설치
- [ ] 각 브라우저에서 개발자 도구 활성화
