Easy-Epidemiology Web 사용설명서

1. 프로그램 소개

Easy-Epidemiology Web (v1.4)는 역학 조사를 위한 데이터 입력, 분석, 시각화 및 보고서 생성을 지원하는 웹 기반 통합 도구입니다.

이 프로그램은 복잡한 역학 데이터를 손쉽게 관리하고, 다양한 통계 분석을 통해 의미 있는 결과를 도출할 수 있도록 설계되었습니다. 주요 기능은 다음과 같습니다.

  • 직관적인 표 형식의 데이터 입력 및 관리
  • 환자 특성, 임상 증상 등 자동 분석 및 시각화
  • 유행 곡선, 잠복기 차트 등 동적 차트 생성
  • 환자-대조군(Odds Ratio) 및 코호트(Relative Risk) 연구 지원
  • 분석 결과를 바탕으로 HWPX 보고서 초안 자동 생성

2. 프로그램 실행방법

이 프로그램은 별도의 복잡한 설치 과정 없이, 압축 해제 후 바로 사용할 수 있도록 제작되었습니다. 아래의 순서에 따라 프로그램을 실행해주세요.

✅ 실행 순서

  1. 압축 파일 해제:
    전달받은 Easy-Epidemiology Web.zip 압축 파일을 원하는 위치(예: 바탕화면, 내 문서 등)에 해제합니다.
  2. 폴더 이동:
    압축 해제가 완료되면 생성된 Easy-Epidemiology Web 폴더로 들어갑니다.
  3. 실행 파일 실행:
    폴더 내에 있는 Easy-Epidemiology-Launcher.html 파일을 찾아 더블클릭하여 실행합니다.
  4. 프로그램 확인:
    잠시 후, 컴퓨터의 기본 웹 브라우저(Chrome, Edge 등)가 자동으로 열리면서 프로그램 화면이 나타납니다.

중요 주의사항

프로그램이 정상적으로 동작하려면, 압축 해제한 폴더 내의 어떤 파일이나 하위 폴더의 이름, 위치를 절대로 변경해서는 안 됩니다. 파일이나 폴더 구조가 변경될 경우, 프로그램이 실행되지 않거나 심각한 오류가 발생할 수 있습니다.

3. 데이터 입력 탭 사용 설명

이 탭은 역학조사 데이터를 입력하고 관리하는 핵심 공간입니다. 엑셀과 유사한 표 형태로 되어 있어 데이터를 쉽고 효율적으로 다룰 수 있습니다.

3.1. 화면 구성 이해하기

"데이터 입력" 탭에 접속하시면 다음과 같은 주요 영역을 보실 수 있습니다.

  • ① 데이터 입력 표 (그리드)
    • 화면의 가장 큰 부분을 차지하며, 실제 역학조사 데이터를 입력하고 수정하는 공간입니다. 엑셀 시트처럼 행(가로줄)과 열(세로줄)로 구성되어 있습니다.
  • ② 상단 기능 버튼 (Function Bar)
    • 데이터 입력 표 위에 위치한 다양한 버튼들입니다. 행 추가, 열 삭제, 엑셀 파일 가져오기/내보내기 등 데이터 관리에 필요한 모든 기능이 이곳에 모여 있습니다.
  • ③ 고정된 주요 항목
    • 표의 가장 왼쪽에 위치한 몇몇 열은 역학조사에서 항상 필요한 중요한 항목들입니다.
    • '환자 여부': 해당 행의 대상이 환자인지 아닌지를 표시합니다. (예: '1' 또는 '0'으로 입력)
    • '증상 발현일': 증상이 나타난 날짜와 시간을 입력하는 칸입니다. (예: 'YYYY-MM-DD HH:MM' 형식)
    • '개별 노출 시간': (필요시) 개별적인 노출 시간을 입력하는 칸입니다.
    • '확진자 여부': (필요시) 해당 대상의 확진 여부를 입력하는 칸입니다.
3.2. 컨텍스트 메뉴 (우클릭 메뉴) 기능

표의 셀을 마우스 오른쪽 버튼으로 클릭하면 컨텍스트 메뉴가 나타납니다. 이 메뉴는 클릭한 위치와 선택된 영역에 따라 다양한 옵션을 제공합니다.

셀 관련 메뉴
  • 셀 데이터 삭제: 선택된 셀의 데이터만 지웁니다. 여러 셀이 선택된 경우 모든 선택된 셀의 데이터가 지워집니다.
행 관련 메뉴
  • 위에 행 삽입: 선택된 행 위에 새로운 빈 행을 추가합니다.
  • 아래에 행 삽입: 선택된 행 아래에 새로운 빈 행을 추가합니다.
  • 행 데이터 지우기: 선택된 행의 모든 데이터를 지웁니다.
  • 행 삭제: 선택된 행을 완전히 삭제합니다.
열 관련 메뉴
  • 왼쪽에 열 삽입: 선택된 열 왼쪽에 새로운 빈 열을 추가합니다.
  • 오른쪽에 열 삽입: 선택된 열 오른쪽에 새로운 빈 열을 추가합니다.
  • 열 데이터 삭제: 선택된 열의 모든 데이터를 지웁니다.
  • 열 삭제: 선택된 열을 완전히 삭제합니다 (최소 1개 열은 유지됩니다).
특수 메뉴
  • 빈 행 삭제: 연번 열에서 우클릭 시 나타나는 메뉴로, 데이터가 없는 빈 행들을 모두 삭제합니다.
3.3. 데이터 필터링 기능

데이터 입력 표에서 특정 조건을 만족하는 데이터만 표시하여 필요한 정보를 빠르게 찾을 수 있습니다. 필터 기능은 별도의 필터 버튼이 없고, 헤더 셀을 우클릭하여 나타나는 컨텍스트 메뉴를 통해 사용합니다.

필터 사용 방법

단계별 안내

  1. 컨텍스트 메뉴 열기: 필터를 적용하고 싶은 열의 헤더 셀(제목 부분)을 마우스 오른쪽 버튼으로 클릭합니다.
  2. 필터 값 확인: 나타나는 컨텍스트 메뉴에서 해당 열에 존재하는 모든 고유한 값들이 체크박스 형태로 표시됩니다. 각 값 옆에는 해당 값이 몇 개 있는지 개수가 표시됩니다.
  3. 필터 값 선택: 보이고 싶은 값들의 체크박스를 클릭하여 체크합니다. 체크된 값들에 해당하는 데이터만 표에 표시됩니다.
  4. 필터 적용 확인: 필터가 적용되면 해당 열 헤더에 필터 아이콘(🔽)이 표시되어 필터가 활성화되었음을 알 수 있습니다.
필터 지원 컬럼 종류

다음과 같은 종류의 컬럼에서 필터 기능을 사용할 수 있습니다:

  • 환자 여부 컬럼: '1'(환자), '0'(비환자), '빈 셀' 등으로 필터링
  • 확진자 여부 컬럼: 확진 여부에 따른 필터링
  • 임상 증상 컬럼: 각 증상별 발현 여부에 따른 필터링
  • 식단 정보 컬럼: 섭취한 음식별 필터링
  • 기본 정보 컬럼: 성별, 연령대, 지역 등 기본 정보별 필터링
  • 날짜/시간 컬럼: 증상 발현일, 개별 노출 시간 등 날짜별 필터링
필터 해제 방법

단계별 안내

  1. 개별 필터 해제: 필터가 적용된 열의 헤더 셀을 마우스 오른쪽 버튼으로 클릭한 후, 컨텍스트 메뉴에서 체크를 해제하고 싶은 값의 체크박스를 클릭하여 해제합니다.
  2. 모든 필터 해제: 필터가 적용된 상태에서는 컨텍스트 메뉴 맨 아래에 "모든 필터 해제" 옵션이 나타납니다. 이를 클릭하면 모든 필터가 한 번에 해제됩니다.
필터 사용 팁
  • 다중 필터: 여러 열에 동시에 필터를 적용할 수 있습니다. 예를 들어, '환자 여부'에서 '1'을 선택하고, '성별'에서 '여성'을 선택하면 여성 환자만 표시됩니다.
  • 빈 셀 필터링: 데이터가 입력되지 않은 빈 셀들도 별도로 필터링할 수 있습니다. 컨텍스트 메뉴에서 '빈 셀' 옵션을 확인해보세요.
  • 필터 상태 확인: 필터가 적용된 열의 헤더에는 필터 아이콘이 표시되어 어떤 열에 필터가 적용되어 있는지 쉽게 확인할 수 있습니다.
3.4. 상단 기능 버튼 (Function Bar) 자세히 알아보기

데이터 입력 표 위에 있는 버튼들은 여러분의 작업을 도와주는 도구들입니다. 각 버튼의 역할과 사용법을 자세히 알아봅시다.

셀 정보 및 입력창
  • 셀 주소 표시: 현재 선택된 셀의 위치가 표시됩니다 (예: 'A1', 'B3').
  • 값 입력창: 선택된 셀의 값을 직접 수정할 수 있습니다. 입력한 값은 실시간으로 셀에 반영됩니다.
실행 취소/다시 실행 버튼
  • 실행 취소 (Undo): Ctrl+Z와 동일한 기능으로, 이전 작업을 되돌립니다.
  • 다시 실행 (Redo): Ctrl+Y와 동일한 기능으로, 취소한 작업을 다시 실행합니다.
필터 상태 표시
  • 필터 적용됨 배지: 필터가 적용된 경우 "필터 적용됨" 배지가 표시되며, 현재 표시되는 행 수와 전체 행 수가 표시됩니다.
  • 필터 관리: 필터 버튼을 클릭하면 모든 필터를 한 번에 해제할 수 있습니다.
열 토글 버튼
  • 확진자 여부: 확진자 여부 열을 표시하거나 숨깁니다. 엑셀 파일에 확진자 정보가 포함된 경우 자동으로 표시됩니다.
  • 개별 노출시간: 개별 노출시간 열을 표시하거나 숨깁니다. 엑셀 파일에 개별 노출시간 정보가 포함된 경우 자동으로 표시됩니다.
데이터 입출력 버튼
  • Excel 파일 가져오기: 컴퓨터에 저장된 Excel 파일을 선택하여 데이터를 가져옵니다.
  • 양식 다운로드: 기본 양식 또는 개별 노출시간 양식을 다운로드합니다.
  • 데이터 내보내기: 현재 입력된 모든 데이터를 Excel 파일로 저장합니다.
시트 편집 버튼
  • 전체 복사: 모든 데이터를 클립보드에 복사합니다.
  • 빈 열 삭제: 데이터가 없는 빈 열들을 모두 삭제합니다.
  • 전체 초기화: 모든 데이터와 설정을 초기화하여 빈 시트로 되돌립니다.
3.5. 하단 행 추가 컨트롤

표 하단에는 행을 추가하고 빈 행을 삭제할 수 있는 컨트롤이 있습니다.

행 추가 방법

단계별 안내

  1. 하단의 숫자 입력창에 추가할 행의 개수를 입력합니다 (기본값: 10개).
  2. Enter 키를 누르거나 "하단에" 텍스트를 클릭합니다.
  3. 지정한 개수만큼의 빈 행이 표의 맨 아래에 추가됩니다.
빈 행 삭제
  • 빈 행 삭제 버튼: 데이터가 입력되지 않은 빈 행들을 모두 자동으로 삭제합니다.
  • 필터 적용 중 제한: 필터가 적용된 상태에서는 행 추가가 제한됩니다.
3.6. 헤더 열 추가/삭제 버튼

각 그룹 헤더(기본정보, 임상증상, 식단)의 오른쪽 상단에 있는 +/- 버튼을 사용하여 열을 추가하거나 삭제할 수 있습니다.

열 추가

단계별 안내

  1. 원하는 그룹 헤더(기본정보, 임상증상, 식단)의 오른쪽 상단에 있는 + 버튼을 클릭합니다.
  2. 해당 그룹의 맨 오른쪽에 새로운 빈 열이 추가됩니다.
열 삭제

단계별 안내

  1. 원하는 그룹 헤더의 오른쪽 상단에 있는 - 버튼을 클릭합니다.
  2. 해당 그룹의 맨 오른쪽 열이 삭제됩니다.
3.7. 데이터 유효성 검사 및 오류 확인

프로그램은 사용자가 올바른 형식의 데이터를 입력하도록 돕는 유효성 검사 기능을 제공합니다.

  • 🔴 오류 표시: 날짜 형식에 맞지 않는 데이터, 숫자만 입력해야 하는 칸에 문자를 입력하는 등 잘못된 데이터가 입력되면, 해당 셀(칸)이 빨간색 테두리로 표시됩니다.
  • 💬 오류 메시지 확인: 빨간색으로 표시된 셀에 마우스 커서를 올리면, 어떤 점이 잘못되었는지에 대한 상세한 오류 메시지가 나타납니다. 메시지를 확인하고 올바른 형식으로 수정해주세요.
💾 3.8. 자동 저장 기능 안내

"데이터 입력 표"에 입력하는 모든 내용은 사용자의 웹 브라우저에 자동으로 저장됩니다.

  • 따라서, 프로그램을 사용하다가 실수로 창을 닫거나 컴퓨터를 재부팅하더라도, 다음에 프로그램을 다시 실행하면 이전에 작업하던 내용이 그대로 남아있습니다.
  • 별도로 '저장' 버튼을 누르지 않아도 데이터가 안전하게 보관되므로, 데이터 손실에 대한 걱정 없이 작업에 집중하실 수 있습니다.
⌨️ 3.9. 키보드 단축키 및 고급 기능

데이터 입력을 더욱 효율적으로 하기 위한 키보드 단축키와 고급 기능들을 알아봅시다.

🎯 기본 키보드 네비게이션
  • 화살표 키: 셀 간 이동 (↑↓←→)
  • Enter: 아래쪽 셀로 이동
  • Tab: 오른쪽 셀로 이동
  • Shift + 화살표: 셀 범위 선택 확장
  • Ctrl + 화살표: 데이터의 끝/시작으로 빠르게 이동
✏️ 편집 모드

4. 환자 특성 탭

입력된 데이터를 바탕으로 환자들의 인구학적, 역학적 특성을 자동으로 요약하고 시각화합니다. 이 탭은 역학조사에서 가장 중요한 기본 분석 기능을 제공합니다.

4.1. 화면 구성 및 기본 정보

환자 특성 탭에 접속하면 다음과 같은 주요 영역을 확인할 수 있습니다.

상단 요약 정보
  • 조사 대상자 수: 데이터 입력 탭에서 입력된 전체 대상자 수를 표시합니다.
  • 총 환자 수: 환자 여부에 '1'이 입력된 대상자 수를 표시합니다.
  • 발병률: (총 환자 수 ÷ 조사 대상자 수) × 100으로 계산된 발병률을 표시합니다. 마우스를 올리면 계산 공식을 확인할 수 있습니다.
  • 확진율: 확진자 여부 열이 활성화된 경우, 확진자 비율을 표시합니다. 마우스를 올리면 계산 공식을 확인할 수 있습니다.
변수 선택 버튼
  • 데이터 입력 탭의 '기본정보' 열들이 버튼으로 표시됩니다.
  • 분석하고 싶은 변수(예: 성별, 연령대, 지역 등)를 클릭하여 선택합니다.
  • 선택된 변수는 파란색 배경으로 강조 표시됩니다.
4.2. 통계표 분석

변수를 선택하면 해당 변수의 상세한 통계 정보가 표 형태로 표시됩니다.

통계표 구성
  • 구분: 선택한 변수의 각 카테고리(예: 남성, 여성)
  • 대상자수: 각 카테고리에 속하는 전체 대상자 수
  • 대상자비율: 전체 대상자 중 해당 카테고리의 비율(%)
  • 환자수: 각 카테고리에서 환자로 분류된 대상자 수
  • 환자비율: 해당 카테고리의 전체 대상자 중 환자의 비율(%)
테이블 복사 기능
  • 통계표 우상단의 "테이블 복사" 버튼을 클릭하면 표 전체가 클립보드에 복사됩니다.
  • 복사된 표는 Excel이나 Word 등 다른 프로그램에 붙여넣기할 수 있습니다.
  • 복사 완료 시 체크 표시가 나타납니다.
라벨 매핑 기능
  • 통계표 하단의 "라벨 매핑" 섹션에서 차트에 표시될 라벨을 사용자가 직접 수정할 수 있습니다.
  • 원본 데이터의 값(예: '1', '2')을 더 의미 있는 라벨(예: '남성', '여성')로 변경할 수 있습니다.
  • 라벨을 수정하면 차트의 X축 라벨이 자동으로 업데이트됩니다.
4.3. 차트 분석 및 커스터마이징

선택한 변수의 분포를 시각적으로 분석할 수 있는 다양한 차트 옵션을 제공합니다.

차트 대상 선택
  • 전체 대상자: 모든 대상자의 분포를 차트로 표시합니다.
  • 환자: 환자로 분류된 대상자만의 분포를 차트로 표시합니다.
데이터 유형 선택
  • 수: 실제 인원수를 막대 높이로 표시합니다.
  • 비율(%): 전체 대비 비율을 막대 높이로 표시합니다.
차트 스타일 커스터마이징

차트 커스터마이징 옵션

  • 폰트 크기: 차트의 모든 텍스트 크기를 조절합니다 (매우 작게 ~ 매우 크게)
  • 차트 너비: 차트의 전체 너비를 조절합니다 (500px ~ 1100px)
  • 막대 너비: 각 막대의 두께를 조절합니다 (30% ~ 70%)
  • 막대 강조: 특정 막대를 강조 표시합니다
    • 강조 없음: 모든 막대를 같은 색상으로 표시
    • 최대값 강조: 가장 큰 값의 막대를 빨간색으로 강조
    • 최소값 강조: 가장 작은 값의 막대를 청록색으로 강조
    • 최대/최소값 강조: 최대값과 최소값을 모두 강조
  • 막대 색상: 막대의 기본 색상을 변경합니다 (9가지 색상 중 선택)
차트 내보내기 및 복사
  • 차트 복사: 차트를 클립보드에 이미지로 복사합니다. 다른 프로그램에 붙여넣기할 수 있습니다.
  • 차트 저장: 차트를 고화질 PNG 이미지 파일로 저장합니다. 파일명은 자동으로 생성됩니다.
4.4. 사용 팁 및 주의사항
효과적인 분석을 위한 팁
  • 변수 선택: 분석 목적에 맞는 변수를 선택하세요. 예를 들어, 성별에 따른 발병률 차이를 보고 싶다면 '성별' 변수를 선택합니다.
  • 차트 대상 활용: '전체 대상자'로 전체 분포를 확인한 후, '환자'로 환자 집단의 특성을 비교 분석하세요.
  • 라벨 매핑 활용: 숫자로 된 데이터(예: 1, 2)를 의미 있는 라벨(예: 남성, 여성)로 변경하여 차트를 더 이해하기 쉽게 만드세요.
  • 강조 기능 활용: 최대값이나 최소값을 강조하여 중요한 패턴을 시각적으로 강조하세요.
주의사항

데이터 해석 시 주의사항

환자비율은 해당 카테고리의 전체 대상자 중 환자의 비율을 의미합니다. 예를 들어, 남성 100명 중 환자 20명이면 환자비율은 20%입니다. 이는 전체 대상자 중 남성 환자의 비율과는 다른 개념입니다.

테이블 복사 기능

  • 폰트 크기: 차트의 모든 텍스트 크기를 조절합니다 (매우 작게 ~ 매우 크게)
  • 차트 너비: 차트의 전체 너비를 조절합니다 (500px ~ 1100px)
  • 막대 너비: 각 막대의 두께를 조절합니다 (30% ~ 70%)
  • 색상: 막대의 기본 색상을 변경합니다 (9가지 색상 중 선택)

    • 최대값 강조: 가장 높은 발현율의 막대를 빨간색으로 강조
    • 최소값 강조: 가장 낮은 발현율의 막대를 청록색으로 강조
    • 최대/최소값 강조: 최대값과 최소값을 모두 강조
  • 정렬: 증상들을 정렬 방식으로 표시합니다

7. 환자대조군 연구

환자대조군 연구(Case-Control Study)는 질병이 발생한 환자군과 질병이 발생하지 않은 대조군을 비교하여 질병의 원인을 찾는 역학 연구 방법입니다. 이 탭에서는 식중독 역학조사에서 의심식품과 질병 간의 연관성을 통계적으로 분석합니다.

7.1. 화면 구성 및 기본 정보

환자대조군 연구 탭에 접속하면 다음과 같은 주요 영역을 확인할 수 있습니다.

화면 레이아웃
  • 상단 영역: 폰트 크기 조절 및 총 분석 대상자 수 표시
  • 분석 결과 영역: 요인별 표 분석 결과 테이블과 통계 검정 방법 범례
데이터 요구사항
  • 데이터 입력 탭에서 '환자 여부'에 '1'(환자) 또는 '0'(대조군)이 입력되어야 합니다.
  • 식단 정보가 입력되어 있어야 요인별 분석이 가능합니다.
  • 환자군과 대조군 모두에 데이터가 있어야 통계 분석이 가능합니다.
7.2. 요인별 표 분석 결과

각 식단 요인에 대해 환자군과 대조군의 섭취 여부를 비교하여 통계적 연관성을 분석합니다.

테이블 구성
  • 요인(식단): 분석 대상 식단 항목명
  • 환자군: 환자군의 섭취자 수, 비섭취자 수, 합계
  • 대조군: 대조군의 섭취자 수, 비섭취자 수, 합계
  • 카이제곱 P-value: 통계적 유의성 검정 결과
  • 오즈비(Odds Ratio): 질병과 요인 간의 연관성 강도
  • 95% 신뢰구간: 오즈비의 신뢰구간 (하한, 상한)
통계 검정 방법

자동 선택되는 검정 방법

  • Fisher의 정확검정: 기대빈도가 5 미만인 셀이 있을 때 자동 적용 (표시: *)
  • 카이제곱 검정: 모든 셀의 기대빈도가 5 이상일 때 적용
  • Yates 보정: 2x2 분할표에서 연속성 보정을 적용할 수 있습니다
결과 해석
  • P-value < 0.05: 통계적으로 유의한 연관성이 있음을 의미합니다 (노란색 배경으로 강조)
  • 오즈비 > 1: 해당 요인이 질병 발생 위험을 증가시킵니다
  • 오즈비 < 1: 해당 요인이 질병 발생 위험을 감소시킵니다
  • 95% 신뢰구간: 1을 포함하지 않으면 통계적으로 유의합니다
7.3. 필터 및 설정 옵션

분석 결과를 필터링하고 통계 검정 방법을 조정할 수 있습니다.

오즈비 필터

필터 사용 방법

  1. 필터 활성화: "OR ≥" 버튼을 클릭하여 필터를 활성화합니다
  2. 임계값 조정: 숫자 버튼을 클릭하여 임계값을 변경합니다 (2.0 → 3.0 → 4.0)
  3. 결과 확인: 설정된 오즈비 이상인 요인들만 테이블에 표시됩니다
  4. 필터 해제: 다시 버튼을 클릭하여 필터를 비활성화합니다
Yates 보정 설정
  • Yates 보정 적용: 2x2 분할표에서 연속성 보정을 적용합니다

신뢰구간 해석
  • 95% 신뢰구간이 1을 포함하지 않음: 통계적으로 유의한 연관성이 있습니다
  • 95% 신뢰구간이 1을 포함함: 통계적으로 유의하지 않을 수 있습니다
  • 신뢰구간이 넓음: 표본 크기가 작거나 변동성이 큽니다

분석 시 주의사항

  • 0인 셀이 있는 경우 Haldane-Anscombe 보정이 자동으로 적용됩니다
  • 결과 해석 시 임상적 의의도 함께 고려해야 합니다

8. 코호트 연구 탭

코호트 연구(Cohort Study)는 특정 요인(예: 의심 식품 섭취)에 노출된 집단과 노출되지 않은 집단의 질병 발생률을 비교하여, 해당 요인이 질병 발생에 미치는 영향을 평가하는 역학 연구 방법입니다. 이 탭에서는 상대위험도(Relative Risk, RR)와 발병률 등 주요 지표를 자동으로 계산하고 시각화합니다.

8.1. 화면 구성 및 기본 정보

코호트 연구 탭에 접속하면 다음과 같은 주요 영역을 확인할 수 있습니다.

화면 레이아웃
  • 상단 영역: 폰트 크기 조절 및 총 분석 대상자 수 표시
  • 분석 결과 영역: 요인별 표 분석 결과 테이블과 통계 검정 방법 범례
데이터 요구사항
  • 데이터 입력 탭에서 '환자 여부'에 '1'(환자) 또는 '0'(비환자)이 입력되어야 합니다.
  • 식단 정보가 입력되어 있어야 요인별 분석이 가능합니다.
  • 노출군(섭취자)과 비노출군(비섭취자) 모두에 데이터가 있어야 통계 분석이 가능합니다.
8.2. 요인별 표 분석 결과

각 식단 요인에 대해 노출군(섭취자)과 비노출군(비섭취자)의 질병 발생률을 비교하여 통계적 연관성을 분석합니다.

테이블 구성
  • 요인(식단): 분석 대상 식단 항목명
  • 노출군(섭취자): 대상자수, 환자수, 발병률(%)
  • 비노출군(비섭취자): 대상자수, 환자수, 발병률(%)
  • 카이제곱 P-value: 통계적 유의성 검정 결과
  • 상대위험도(Relative Risk): 노출군의 질병 발생 위험이 비노출군에 비해 몇 배 높은지
  • 95% 신뢰구간: 상대위험도의 신뢰구간 (하한, 상한)
통계 검정 방법

자동 선택되는 검정 방법

  • Fisher의 정확검정: 기대빈도가 5 미만인 셀이 있을 때 자동 적용 (표시: *)
  • 카이제곱 검정: 모든 셀의 기대빈도가 5 이상일 때 적용
  • Yates 보정: 2x2 분할표에서 연속성 보정을 적용할 수 있습니다
결과 해석
  • P-value < 0.05: 통계적으로 유의한 연관성이 있음을 의미합니다 (노란색 배경으로 강조)
  • 상대위험도(RR) > 1: 해당 요인이 질병 발생 위험을 증가시킵니다
  • 상대위험도(RR) < 1: 해당 요인이 질병 발생 위험을 감소시킵니다
  • 95% 신뢰구간: 1을 포함하지 않으면 통계적으로 유의합니다
8.3. 설정 옵션 및 기능
  • Yates 보정: 2x2 분할표에서 연속성 보정 적용/해제 가능

9. 보고서 작성

이 탭은 각 탭에서 분석된 모든 결과를 종합하여 한글 형식(.hwpx)의 역학조사 보고서 초안을 자동으로 생성하는 마지막 단계입니다. 사용자는 별도로 내용을 입력하는 것이 아니라, 이전 단계들에서 분석하고 저장한 내용을 바탕으로 완성된 보고서를 다운로드할 수 있습니다.

9.1. 화면 구성

'보고서 작성' 탭은 크게 두 가지 영역으로 나뉩니다. 왼쪽은 보고서에 포함될 항목을 확인하는 '보고서 항목' 영역이고, 오른쪽은 생성될 보고서의 내용을 미리 보는 '미리보기' 영역입니다.

  • ① 보고서 항목 (왼쪽): 보고서에 자동으로 채워질 데이터 항목들의 목록을 보여줍니다. 각 항목 옆의 배지를 통해 데이터가 정상적으로 불러와졌는지('입력됨' 또는 실제 값) 또는 해당 데이터가 아직 없는지('미입력') 확인할 수 있습니다.
  • ② 미리보기 (오른쪽): 왼쪽의 항목들이 실제 보고서 형식에 맞춰 어떻게 배치되는지 보여줍니다. 차트와 표, 분석 결과가 포함된 보고서 초안을 실시간으로 확인할 수 있습니다.
  • ③ 다운로드 버튼 (오른쪽 상단): 최종적으로 완성된 보고서를 한글 파일(.hwpx)로 다운로드하는 버튼입니다.

  2. 연구 디자인 선택: '보고서 항목' 목록 상단에서 이번 조사의 설계에 맞는 '조사 디자인'을 선택합니다. (예: 환자-대조군 연구)
  3. 차트 및 감염원 저장 (⭐중요): '유행곡선 및 잠복기 분석' 탭으로 이동하여 보고서에 넣고 싶은 유행곡선과 잠복기 차트를 최종적으로 커스터마이징한 후, 반드시 '보고서 저장' 버튼을 클릭합니다. 추정 감염원도 해당 탭에서 선택해야 합니다. (자세한 방법은 9.3 참고)
  4. 미리보기 확인: 모든 항목이 정상적으로 '입력됨'으로 표시되고, 오른쪽 '미리보기' 화면에 원하는 내용이 모두 보이는지 확인합니다.
  5. 보고서 다운로드: '보고서 다운로드' 버튼을 클릭하여 최종 보고서 파일(.hwpx)을 받습니다.
9.3. 추정 감염원 및 차트 삽입 방법

보고서의 핵심인 '추정 감염원'과 '유행곡선/잠복기 차트'는 '유행곡선 및 잠복기 분석' 탭에서 직접 저장해야 합니다. 이 과정을 거치지 않으면 보고서에 해당 내용이 포함되지 않습니다.

추정 감염원 및 차트 저장하기

단계별 안내

  1. '유행곡선 및 잠복기 분석' 탭으로 이동합니다. (매뉴얼 6번 항목 참고)
  2. 상단의 '추정 감염원' 드롭다운 메뉴에서 보고서에 명시할 의심 식품(감염원)을 선택합니다.
  3. 유행곡선 차트를 원하는 형태로 커스터마이징합니다. 시간 간격, 색상, 폰트 크기 등 보고서에 표시될 최종 디자인으로 설정합니다.
  4. 차트 설정이 끝나면, 유행곡선 차트 우측 상단의 [보고서 저장] 버튼을 클릭합니다.
  5. 잠복기 차트도 동일하게 커스터마이징한 후, 잠복기 차트 우측 상단의 [보고서 저장] 버튼을 클릭합니다.
  6. 다시 '보고서 작성' 탭으로 돌아오면, '유행곡선 차트', '잠복기 차트' 항목이 '입력됨'으로 변경되고 미리보기에 해당 차트가 나타난 것을 확인할 수 있습니다.

꼭 기억하세요!

차트는 현재 화면에 보이는 그대로 이미지로 저장됩니다. 따라서 반드시 원하는 형태로 커스터마이징을 모두 마친 후 '보고서 저장' 버튼을 눌러야 합니다. 버튼을 누르지 않으면 차트는 보고서에 삽입되지 않습니다.

9.4. 미리보기 확인 및 다운로드

모든 준비가 끝나면 오른쪽 '미리보기' 창을 통해 최종 보고서의 내용을 검토할 수 있습니다.

  • 자동 채움: 조사 개요, 발병률, 인구학적 특성, 주요 증상 통계표, 유행곡선, 잠복기 분석, 통계 분석 결과 등 모든 내용이 자동으로 채워집니다.
  • 플레이스홀더: 만약 특정 값이 -- 와 같이 표시된다면, 해당 데이터가 부족하거나 관련 분석이 수행되지 않았음을 의미합니다. 이전 탭으로 돌아가 작업을 완료해주세요.
  • 다운로드: 미리보기에 문제가 없음을 확인했다면, 우측 상단의 [보고서 다운로드] 버튼을 클릭하여 한글(.hwpx) 파일을 다운로드합니다. 이 파일은 추가 편집이 가능합니다.
중요한 제한사항

데이터 행 수 제한

34개 행을 초과하는 데이터의 경우, 보고서의 표4 '요인별 분석결과' 표에 통계 분석 데이터가 포함되지 않습니다. 이는 보고서 템플릿의 형식 제한으로 인한 것으로, 34개 행 이하의 데이터에서만 완전한 통계 분석 결과가 보고서에 포함됩니다.

대용량 데이터를 분석할 때는 이 제한사항을 고려하여 데이터를 분할하거나, 핵심 데이터만 선별하여 분석하는 것을 권장합니다.
