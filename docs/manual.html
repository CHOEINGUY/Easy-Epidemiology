1. 프로그램 소개

Easy-Epidemiology Web (v2.0)는 역학 조사를 위한 데이터 입력, 분석, 시각화 및 보고서 생성을 지원하는 웹 기반 통합 도구입니다.

이 프로그램은 복잡한 역학 데이터를 손쉽게 관리하고, 다양한 통계 분석을 통해 의미 있는 결과를 도출할 수 있도록 설계되었습니다. 주요 기능은 다음과 같습니다.

  • 직관적인 표 형식의 데이터 입력 및 관리
  • 환자 특성, 임상 증상 등 자동 분석 및 시각화
  • 유행 곡선, 잠복기 차트 등 동적 차트 생성
  • 환자-대조군(Odds Ratio) 및 코호트(Relative Risk) 연구 지원
  • 분석 결과를 바탕으로 HWPX 보고서 초안 자동 생성

2. 프로그램 실행방법

이 프로그램은 별도의 복잡한 설치 과정 없이, 압축 해제 후 바로 사용할 수 있도록 제작되었습니다.

✅ 실행 순서

  1. 압축 파일 해제: 전달받은 Easy-Epidemiology Web.zip 압축 파일을 원하는 위치(예: 바탕화면)에 해제합니다.
  2. 폴더 이동: 생성된 Easy-Epidemiology Web 폴더로 들어갑니다.
  3. 실행 파일 실행: Easy-Epidemiology.bat 파일을 찾아 더블클릭합니다.
  4. 프로그램 확인: 웹 브라우저가 열리며 프로그램이 시작됩니다.

중요 주의사항

프로그램 폴더 내의 파일 이름이나 위치를 절대 변경하지 마세요. 프로그램이 작동하지 않을 수 있습니다.

3. 데이터 입력 탭 사용 설명

이 탭은 역학조사 데이터를 입력하고 관리하는 핵심 공간입니다. 엑셀과 유사한 표 형태로 되어 있어 데이터를 쉽게 다룰 수 있습니다.

3.1. 화면 구성 이해하기

"데이터 입력" 탭의 주요 영역은 다음과 같습니다.

  • ① 데이터 입력 표 (그리드): 실제 데이터를 입력하는 공간입니다.
  • ② 상단 기능 버튼 (Function Bar): 엑셀 가져오기, 내보내기 등 주요 기능 버튼이 있습니다.
  • ③ 고정된 주요 항목: '환자 여부', '증상 발현일' 등 필수 입력 열입니다.
3.2. 컨텍스트 메뉴 (우클릭)

셀을 우클릭하면 셀/행/열에 대한 편집 메뉴가 나타납니다.

  • 행/열 추가 및 삭제: 원하는 위치에 행이나 열을 삽입하거나 삭제할 수 있습니다.
  • 데이터 삭제: 선택한 셀이나 행/열의 내용을 지웁니다.
3.3. 데이터 필터링

헤더를 우클릭하여 원하는 데이터만 필터링하여 볼 수 있습니다.

필터 사용 방법

  1. 헤더 셀 우클릭 → 원하는 값 체크 → 필터 적용됨
3.4. 상단 기능 버튼
  • Excel 파일 가져오기: 기존 데이터를 불러옵니다.
  • 데이터 내보내기: 현재 데이터를 저장합니다.
  • 전체 초기화: 모든 데이터를 지우고 초기화합니다.
3.5. 하단 행 추가 컨트롤

표 하단에서 원하는 수만큼 행을 추가하거나 빈 행을 삭제할 수 있습니다.

3.6. 헤더 열 추가/삭제

각 그룹 헤더(임상증상 등) 우측의 +/- 버튼으로 열을 손쉽게 관리하세요.

3.7. 데이터 유효성 검사

잘못된 형식이 입력되면 셀 테두리가 빨간색으로 표시됩니다.

3.8. 자동 저장

안심하세요!

입력하는 모든 내용은 브라우저에 실시간으로 자동 저장됩니다.

3.9. 키보드 단축키

엑셀과 유사한 단축키를 지원합니다 (화살표, Enter, Tab, Ctrl+C/V 등).

4. 환자 특성 탭

환자들의 인구학적 특성을 자동으로 요약하고 시각화합니다.

4.1. 개요

전체 대상자, 환자 수, 발병률 등을 한눈에 확인할 수 있습니다.

4.2. 통계표 분석

성별, 연령대 등 변수를 선택하면 상세 통계표가 생성됩니다.

4.3. 차트 분석

막대 그래프로 데이터를 시각화하며, 색상 및 너비 등을 커스터마이징할 수 있습니다.

4.4. 팁 및 주의사항

해석 주의

환자 비율과 발병률의 차이를 구분하여 해석하시기 바랍니다.

5. 임상 증상 탭

환자들의 주요 증상 호소율을 분석합니다.

5.1. 개요

등록된 모든 증상에 대한 빈도와 비율을 분석합니다.

5.2. 통계표 분석

증상별 환자 수와 백분율을 표로 제공합니다.

5.3. 차트 분석

증상 빈도를 막대 차트로 시각화합니다. 가로/세로 모드를 지원합니다.

5.4. 분석 방법

분석 팁

'내림차순' 정렬을 사용하면 가장 흔한 증상을 쉽게 파악할 수 있습니다.

6. 유행곡선 및 잠복기 분석

시간별 환자 발생 패턴을 분석하여 감염원을 추정합니다.

6.1. 개요

유행곡선(Epi Curve)과 잠복기 분석을 동시에 수행할 수 있습니다.

6.2. 추정 감염원 선택

통계적으로 유의한 위험 요인을 차트에 표시할 수 있습니다.

6.3. 증상 발현 분석

시간대별 환자 발생 수를 집계합니다.

6.4. 유행곡선 차트

시간 간격을 조절하며 유행 패턴을 시각적으로 확인할 수 있습니다.

6.5. 잠복기 분석

노출 시점을 설정하면 최소/최대/평균 잠복기가 자동 계산됩니다.

6.6. 잠복기 차트

잠복기 분포를 히스토그램으로 확인합니다.

6.7. 분석 방법

단일 노출인지 지속 노출인지 곡선의 형태를 통해 유추해보세요.

7. 환자대조군 연구

환자군과 대조군의 노출력 비교를 통해 위험 요인을 규명합니다 (주로 식중독 조사).

7.1. 개요

오즈비(OR)와 P-value를 자동 계산합니다.

7.2. 표 분석 결과

각 식품별 섭취율 비교 및 통계 검정 결과를 보여줍니다.

7.3. 필터 및 설정

유의한 항목(예: OR > 2.0)만 필터링하여 볼 수 있습니다.

7.4. 복사 기능

분석 표를 그대로 복사하여 보고서에 붙여넣을 수 있습니다.

7.5. 통계 검정 방법

카이제곱 검정 및 Fisher의 정확 검정을 상황에 맞춰 자동 적용합니다.

7.6. 결과 해석

해석 주의

P-value가 0.05 미만이고 OR이 1보다 크면 통계적으로 유의한 위험 요인일 가능성이 높습니다.

7.7. 사용 예시

여러 식품 중 가장 OR이 높고 신뢰구간이 1을 포함하지 않는 식품을 의심 식품으로 지목하세요.

8. 코호트 연구

노출군과 비노출군의 발병률 비교를 통해 위험도를 산출합니다 (상대위험도 RR).

8.1. 개요

전체 모집단이 파악된 경우 사용합니다.

8.2. 분석 결과

식품별 RR(상대위험도) 및 95% 신뢰구간을 제공합니다.

8.3. 설정

Yates 보정 등을 선택할 수 있습니다.

8.4. 통계 방법

기대 빈도에 따라 적절한 검정법이 자동 적용됩니다.

8.5. 결과 해석

RR > 1 인 경우 해당 식품 섭취 시 발병 위험이 높음을 의미합니다.

8.6. 사용 예시

학교 급식 등 전체 식단과 명단이 확보된 사례에서 유용합니다.

9. 보고서 작성

분석된 모든 결과를 종합하여 HWPX 포맷의 보고서를 자동 생성합니다.

9.1. 개요

왼쪽의 항목 리스트와 오른쪽의 미리보기 화면으로 구성됩니다.

9.2. 생성 단계
  1. 모든 분석 탭의 내용 확인
  2. '유행곡선' 탭에서 차트 '보고서 저장' 클릭 (필수!)
  3. 미리보기 검토 후 다운로드
9.3. 차트 삽입 방법

중요

차트는 [보고서 저장] 버튼을 눌러야만 보고서에 포함됩니다.

9.4. 미리보기 및 다운로드

최종 확인 후 .hwpx 파일로 내려받아 한글 프로그램에서 편집하세요.

10. 사례군 조사

특정 사례들에 대한 심층적인 정보를 리스트 형태로 요약 제공합니다.
