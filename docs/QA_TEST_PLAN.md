# Easy-Epidemiology Web QA 테스트 계획

## 1. 프로젝트 개요

### 1.1 프로젝트 정보
- **프로젝트명**: Easy-Epidemiology Web v1.0
- **기술 스택**: Vue.js 3, Vuex, Vuetify, Chart.js, ECharts, XLSX
- **주요 기능**: 역학조사 데이터 입력, 분석, 시각화
- **테스트 환경**: Windows 10, Chrome/Firefox/Edge

### 1.2 주요 기능 모듈
1. **데이터 입력 (DataInputVirtual)**: 가상 스크롤 기반 대용량 데이터 입력
2. **환자특성 (PatientCharacteristics)**: 대상자 특성 분포 분석 및 차트
3. **유행곡선 (EpidemicCurve)**: 증상 발현 시간별 유행곡선 및 잠복기 분석
4. **임상증상 (ClinicalSymptoms)**: 임상증상별 빈도 분석
5. **환자대조군 (CaseControl)**: 오즈비(OR) 분석
6. **코호트 (CohortStudy)**: 상대위험비(RR) 분석
