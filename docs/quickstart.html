Easy-Epidemiology Quick Start

menu_book 전체 매뉴얼

rocket_launch 실행 방법

1

압축 해제

다운로드 받은 Easy-Epidemiology Web.zip 파일의 압축을 풉니다.

2

실행 파일 클릭

폴더 내의 Easy-Epidemiology.bat 파일을 실행하면 브라우저가 자동으로 열립니다.

warning

주의: 폴더 내의 파일 이름이나 위치를 변경하면 프로그램이 실행되지 않을 수 있습니다.

grid_on 데이터 입력

  • content_paste 엑셀 호환

    Ctrl+C, Ctrl+V로 엑셀 데이터를 그대로 붙여넣을 수 있습니다.

  • upload_file 드래그 앤 드롭

    엑셀 파일을 화면 위로 끌어다 놓으면 자동으로 업로드됩니다.

  • save 자동 저장

    입력하는 모든 내용은 브라우저에 실시간으로 안전하게 자동 저장됩니다.

  • error_outline 오류 알림

    날짜 형식이나 필수 값이 누락되면 빨간색 테두리로 즉시 알려줍니다.

analytics 분석 기능

환자 특성 & 임상 증상

입력된 데이터를 바탕으로 성별, 연령, 지역별 통계와 증상 발현율을 자동으로 계산하고 시각화합니다.

역학 조사 (유행곡선 등)

환자-대조군 연구, 코호트 연구를 위한 OR(오즈비), RR(상대위험비), P-value를 자동으로 산출합니다.
