# Easy-Epidemiology v1.0 1차 피드백 정리
**날짜**: 2025년 7월 4일

## 🚨 긴급 수정 필요 사항

### 1. 데이터 가져오기 오류
- **문제**: 템플릿에서 빈 열이 있을 때(중간에) 에러 발생
- **우선순위**: 높음
- **상태**: 수정 필요

### 2. 차트 저장/복사 기능 문제
- **문제**: 차트 너비를 넓게 설정해도 화면이 좁을 때 그대로 가져오는 문제
- **요구사항**: 화면과 상관없이 차트 이미지가 추출되도록 수정
- **영향 기능**: 차트 저장하기, 차트 복사 기능
- **우선순위**: 높음

### 3. Undo/Redo 기능 오류
- **문제**: 버튼 사용 시 바디테이블 데이터 전체 삭제되는 에러
- **우선순위**: 높음

### 4. 환자특성 비율(%) 차트 오류
- **문제**: 차트에서 문제 발생
- **우선순위**: 높음

### 5. 양식파일 첨부파일 문제
- **문제**: 첨부파일이 안되는 문제
- **우선순위**: 높음

## 🎨 UI/UX 개선 사항

### 1. 폰트 크기 설정 개선
- **현재**: 포인트(pt) 단위 사용
- **문제**: 사용자가 pt와 혼동할 수 있음
- **제안**: "작게", "중간", "크게" 옵션으로 변경

### 2. 차트 범례 위치 개선
- **문제**: 범례 위치가 애매함, X축처럼 느껴짐
- **상태**: 고민 필요

## 📊 유행곡선 차트 개선 사항

### 1. 시간 표현 수정
- **현재**: 12시~24시로 표현
- **요구사항**: 12시~23시59분으로 수정 (실무 기준)

### 2. Y축 여백 개선
- **문제**: 최대값이 10일 때 너무 타이트해 보임
- **요구사항**: Y축 최대값에 여유 공간 추가

### 3. 확진자 꺾은선 기능 (아이디어)
- **제안**: 데이터입력에 "확진여부" 열 추가
- **위치**: 환자여부 열 옆에 확진여부 열 추가

### 4. 의심식단 표시 기능 (아이디어)
- **제안**: 차트 내부에 풋노트로 의심식단 표시
- **목적**: 문제가 되는 식단에 대한 예상 의심식단 안내

### 5. 날짜/기울기체 선택 기능 (아이디어)
- **제안**: 날짜, 기울기체와 기존 상태를 선택하는 기능 추가

## 📈 임상증상 차트 개선

### 1. 정렬 기능 추가 (아이디어)
- **제안**: 오름차순/내림차순 정렬 기능

## 📋 환자대조군/코호트 테이블 개선

### 1. 풋노트 복사 기능 (아이디어)
- **제안**: 테이블 복사 시 풋노트도 함께 클립보드에 복사

### 2. OR값 필터 기능 (아이디어)
- **제안**: OR값에 대한 필터(사용자 지정 2, 3, 4)로 선택 가능하게

### 3. 0.5 보정 계산 로직 (아이디어)
- **제안**: 0.5를 더해서 계산하는 보정 로직 추가

## 🔍 데이터 입력 및 필터링 개선

### 1. 열 데이터 필터 기능 (아이디어)
- **제안**: 열에서 데이터 필터 기능
- **확장**: 식단에서도 필터 적용

### 2. 사례군조사 탭 추가 (아이디어)
- **제안**: 사례군조사(All 환자) 탭 별도 생성
- **확장**: 식단별발병률 탭 별도 생성

## 🏥 환자특성 개선 사항

### 1. 발병률 퍼센티지 표시
- **현재**: 조사대상자수 100명, 총 환자수 24명
- **요구사항**: 발병률 퍼센티지 표시 추가

### 2. 확진자 정보 반영
- **요구사항**: 데이터입력 창의 환자여부 열에서 확진 수 정보를 환자특성 탭에 반영

### 3. 차트 형태 다양화 (아이디어)
- **제안**: 임상증상/환자특성 차트에서 숫자뿐만 아니라 레인지로도 표시 가능하게

## 🎯 시스템 모드 선택 기능 (아이디어)

### 1. 초기 모드 선택
- **제안**: 첫 창을 열었을 때 수인성/호흡기 모드 선택 기능
