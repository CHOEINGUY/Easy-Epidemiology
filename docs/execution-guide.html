프로그램 실행 방법

설치 없이 바로 실행하는 Easy-Epidemiology Web

folder_zip

1. 압축 파일 해제

다운로드 받은 Easy-Epidemiology Web.zip 파일을 바탕화면이나 내 문서 등 원하는 위치에 압축 해제합니다.

folder_open

2. 폴더 이동

생성된 Easy-Epidemiology Web 폴더 안으로 들어갑니다.

play_arrow

3. 실행 파일 클릭

폴더 내의 Easy-Epidemiology.bat 파일을 찾아 더블클릭합니다.

.bat 확장자가 보이지 않을 수 있습니다. 아이콘을 확인하세요.
priority_high

절대 주의사항

압축을 해제한 폴더 내부의 파일 이름이나 위치를 절대로 변경하지 마세요.
임의로 파일을 삭제하거나 이동하면 프로그램이 정상 작동하지 않습니다.
