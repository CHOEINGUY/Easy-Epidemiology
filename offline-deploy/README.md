# Easy-Epidemiology Web - 오프라인 버전

## 📋 사용법

### 🚀 실행 방법
1. `index.html` 파일을 웹 브라우저에서 열기
2. 또는 로컬 웹 서버를 사용하여 실행

### 🌐 로컬 웹 서버 실행 (권장)
```bash
# Python 3 사용
python -m http.server 8080

# 또는 Node.js 사용
npx serve .

# 또는 PHP 사용
php -S localhost:8080
```

### 📁 파일 구조
- `index.html` - 메인 애플리케이션
- `js/` - JavaScript 파일들
- `css/` - 스타일시트 파일들
- `*.xlsx` - 템플릿 파일들
- `*.zip` - 리포트 템플릿들

### ⚠️ 주의사항
- 이 버전은 오프라인에서 작동합니다
- 로그인 기능이 비활성화되어 있습니다
- 데이터는 브라우저 로컬 스토리지에 저장됩니다
- 관리자 패널이 비활성화되어 있습니다

### 🔧 기능
- 데이터 입력 및 관리
- 통계 분석
- 리포트 생성
- Excel 파일 업로드/다운로드
- 차트 및 그래프 생성
