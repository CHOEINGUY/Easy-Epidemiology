Easy-Epidemiology Web
오프라인 사용 설명서

(보건소·역학조사 담당자용)

이 문서는 컴퓨터를 잘 다루지 않아도 Excel 작업 정도만 할 수 있는 분들을 위해 작성되었습니다. 화면 위치나 버튼 이름은 프로그램 업데이트에 따라 약간 달라질 수 있습니다.

1. 준비 사항

  • ✓프로그램 폴더(예: Easy-Epidemiology Web v1.0)를 한 곳(바탕화면 추천)에 그대로 복사해 둡니다. 압축을 푼 뒤에는 폴더 안의 파일이나 폴더를 절대 건드리지 마세요. (폴더 안에 여러 파일과 폴더가 보이더라도, 아무것도 삭제하거나 옮기지 않아야 정상 작동합니다.)
  • ✓Windows 10 이상 PC, 인터넷 연결은 필수 아님.
  • ✓권장 브라우저: Chrome, Edge, Firefox (2020년 이후 최신 버전).

2. 실행 방법 (가장 쉬운 방법)

  1. 폴더 열기 → 그 안의 index.html 아이콘을 두 번 클릭합니다.
  2. 잠시 후 "Easy-Epidemiology Web" 화면이 뜨면 준비 완료!
Tip: 인터넷이 꺼져 있어도 정상 작동합니다. 새 창이 뜨지 않거나, 화면이 하얗게만 보이면 8. 자주 묻는 질문 항목을 확인하세요.

3. 템플릿 내려받기 & 양식 살펴보기

프로그램 우측 상단의 "다운로드" 아이콘(📥)을 클릭하면 두 가지 Excel 템플릿 중 하나를 저장할 수 있습니다. 어떤 파일을 써야 할지 헷갈리면 아래 설명을 참고하세요.

① 표준 템플릿 (Easy-Epidemiology_Template.xlsx)

  • 가장 많이 사용하는 기본 양식입니다.
  • 필수 열: 환자여부(0=정상, 1=환자), 식단 요인, 증상 등
  • 날짜·시간 열은 서식이 "yyyy-mm-dd hh:mm" 으로 이미 설정되어 있습니다.

② 개별 노출시간 템플릿 (Easy-Epidemiology_Individual_Exposure_Time_Template.xlsx)

  • 같은 장소라도 사람마다 노출시간이 달랐을 때 사용합니다.
  • 의심원 노출시간 열이 추가되어 있습니다. 각 환자가 의심원(식품·장소 등)에 노출된 시각을 입력합니다.
  • 입력 형식: yyyy-mm-dd hh:mm (예: 2023-05-01 22:15)
  • 해당 열을 비워 두면 기존 방식(공통 노출 시점)으로 분석됩니다.
중요: 셀 서식(열 이름, 순서)을 변경하면 프로그램이 파일을 읽지 못할 수 있습니다.

4. 데이터 저장 & 불러오기 (백업/복원)

자동저장 안내: 입력하거나 수정한 데이터는 브라우저에 자동 저장되어, 새로고침하거나 실수로 창을 닫아도 그대로 복원됩니다.
주의! 브라우저의 캐시, 쿠키, 사이트 데이터를 삭제하면 자동저장된 데이터도 함께 사라집니다. 중요한 데이터는 반드시 [데이터 내보내기]로 백업해 두세요.

1. 데이터 내보내기 (엑셀로 저장/백업)

  • [데이터 내보내기] 버튼을 클릭하면 현재 표가 Excel 파일로 저장됩니다.
  • 파일명은 Easy-Epi_exported_data_YYYY-MM-DD HH-mm.xlsx 형식으로 자동 생성되어, 언제 저장한 파일인지 바로 알 수 있습니다.
  • 저장한 파일은 백업, 이동, 동료와 공유 등에 사용할 수 있습니다.
예시: Easy-Epi_exported_data_2024-06-08 15-30.xlsx

2. 데이터 불러오기 (복원/이어하기)

  • [데이터 가져오기] 버튼을 클릭해 저장해둔 파일을 선택하세요.
  • 또는, 파일을 프로그램 창 위로 끌어다 놓기만 해도 불러와집니다.
  • 불러온 데이터는 즉시 표로 나타나며, 이어서 작업할 수 있습니다.

3. 데이터 저장/복원 흐름 & 활용 팁

  1. 작업 중간에 저장: 자동저장 + [데이터 내보내기]로 파일 백업 → 파일명 확인 → 안전하게 보관
  2. 나중에 이어서: [데이터 가져오기] → 파일 선택 → 이전 상태 그대로 복원
  3. 브라우저 캐시/쿠키 삭제 전: 반드시 [데이터 내보내기]로 파일 백업
꼭 기억하세요! 내보낸 파일을 엑셀에서 직접 수정하거나, 열 이름/순서를 바꾸면 불러올 때 오류가 날 수 있습니다. 중요한 데이터는 꼭 내보내기로 백업해두세요.

5. 데이터 수정 & 주요 기능

  • 셀 편집: 셀을 두 번 클릭하거나 Enter → 수정 → Enter 저장 / Esc 취소.
  • 날짜·시간 입력: 날짜·시간 열 더블클릭 → 달력·시계 팝업에서 선택.
  • 행·열 추가/삭제: 표 오른쪽·아래 (+) 버튼 또는 우클릭 메뉴로 삽입/삭제.
  • 드래그 & 드롭: 연번(왼쪽 첫 열)을 잡아 끌어 행 순서를 변경할 수 있습니다.
  • 컨텍스트 메뉴: 셀 우클릭 → 복사, 붙여넣기, Fill Down, 빈 행/열 삽입·삭제 등.
  • 열 토글: [노출시간 열 표시] 스위치로 의심원 노출시간 열 숨김/표시.
  • 개별 노출시간 입력: 의심원 노출시간 열에 각 환자의 노출 시각을 입력 → 잠복기 계산에 사용.
  • 시트 초기화: ⚠️ [시트 초기화] 버튼을 누르면 모든 데이터가 지워집니다.
  • 되돌리기/다시 실행: Ctrl+Z / Ctrl+Shift+Z 또는 Ctrl+Y (최대 100단계).
  • 자동 저장: 수정 내용은 0.3초 이내 브라우저 로컬 저장소에 저장되어 새로고침 후에도 복원됩니다.
  • 토스트 알림: 저장 완료, 오류, 확인 요청 등이 우측 상단 토스트로 표시됩니다.

개인정보 보호

주민번호 등 민감정보는 입력하지 마세요.

7. 분석 결과 보기

화면 좌측 메뉴(또는 상단 탭)에서 원하는 분석을 고를 수 있습니다. 결과 표는 우측 상단 “테이블 복사” 버튼으로 Word/한글에 바로 붙여넣기 가능합니다.

A) 환자 특성 요약 (Patient Characteristics)
  • 성별, 연령대, 기저질환 등 기본정보를 자동 집계합니다.
  • 집계표와 함께 막대·원형 그래프가 제공되어 분포를 한눈에 파악할 수 있습니다.
B) 유행 곡선 (Epidemic Curve)
  • 증상발현시각 데이터를 기반으로 시간대별 막대그래프를 그립니다.
  • 시간 간격을 1h·2h·6h·1d로 바로 변경할 수 있습니다.
  • 상단 [⬇] 아이콘을 눌러 PNG 이미지로 저장 가능합니다.
C) 임상증상 현황 (Clinical Symptoms)
  • 증상별 ‘0/1’ 입력값을 활용해 발생률(%)을 계산합니다.
  • 발생률 ≥ 20% 증상은 자동 강조 표시됩니다.
D) 환자-대조군 연구 (Case-Control)
  • 식단 요인별 2×2 빈도표를 자동 생성합니다.
  • 도출 지표: Odds Ratio(OR), 95% 신뢰구간, P-value
  • 검정 방법: 기대빈도 < 5 → Fisher 정확 검정(*), 기대빈도 ≥ 5 → Yates 보정 카이제곱(-)
  • [OR ≥ 2.0] 버튼으로 유의 요인만 필터링할 수 있습니다.
E) 코호트 연구 (Cohort Study)
  • 발병률(Attack Rate)과 상대위험도(RR)를 자동 계산합니다.
  • 신뢰구간(95% CI) 및 P-value가 함께 표시됩니다.

8. 자주 묻는 질문(FAQ)

A) 화면이 안 뜨고 하얗게만 보여요

→ index.html 을 다른 폴더로 옮기지 않았는지 확인.
→ 크롬 주소창이 "file:///.../index.html" 형태인지 확인.
→ Edge/Safari 일부 버전은 보안 설정 때문에 막힐 수 있으니 Chrome을 권장합니다.

B) Excel 업로드가 10%에서 멈춰요

→ 파일 크기가 10MB를 넘는지 확인 (가능하면 5MB 이하로 나누어 업로드).
→ 열 이름이나 순서를 변경하지 않았는지 확인.

C) 입력한 데이터가 사라졌어요

→ 브라우저 쿠키/캐시를 삭제하면 함께 지워집니다. 중요한 데이터는 [엑셀로 내보내기] 버튼으로 저장해 두세요.

9. 안전하게 종료하기

  1. 필요한 경우 [엑셀로 내보내기] 로 백업 파일을 저장합니다.
  2. 브라우저 탭을 닫으면 종료됩니다. 별도 설치·제거 과정이 없습니다.

10. 기타 유용한 기능

A) 데이터 입출력 & 백업
  • Excel 업로드: [데이터 가져오기] 버튼 또는 드래그&드롭으로 .xlsx 파일을 불러옵니다.
  • 양식 다운로드: Function Bar → [양식 다운로드] → 기본 / 개별 노출시간 중 선택.
  • 데이터 내보내기: [데이터 내보내기] 버튼으로 현재 시트를 Excel(.xlsx)로 저장.
    ※ 저장되는 파일명은 Easy-Epi_exported_data_YYYY-MM-DD HH-mm.xlsx 형식으로 자동 생성됩니다.
    예시: Easy-Epi_exported_data_2024-06-08 15-30.xlsx
    파일명에 날짜와 시간이 포함되어 언제 내보낸 데이터인지 쉽게 구분할 수 있습니다.
  • 자동 저장 & 복구: 수정 내용은 0.3초 이내 브라우저 로컬 저장소에 백업 → 탭 재실행 시 자동 복원.
B) 시트 편집 & 생산성
  • 전체 복사: [전체 복사] 버튼으로 헤더 포함 TSV 형식이 클립보드에 복사됩니다.
  • 빈 열 삭제: [빈 열 삭제] 버튼으로 모든 공백 열을 한 번에 제거.
  • 전체 초기화: [전체 초기화] 버튼으로 데이터·설정을 모두 리셋.
  • Undo / Redo: Ctrl+Z, Ctrl+Shift+Z/Ctrl+Y (최대 100단계).
  • 행 드래그 정렬: 연번 열을 드래그해 행 순서를 재배치.
  • 폰트 크기 조절: [폰트 크기] 버튼(12·14·16px)으로 표 글씨 크기 변경.
  • 컨텍스트 메뉴: 셀 우클릭 → 복사, 붙여넣기, 행/열 삽입·삭제, Fill Down 등.
C) 차트 & 결과 활용
  • 표 복사: 각 분석 탭 우측 상단 [테이블 복사] → Word/한글에 바로 붙여넣기.
  • OR ≥ 2.0 필터: Case-Control 탭 [OR ≥ 2.0] 토글로 유의 식단 요인만 표시.
  • 차트 PNG 저장: 모든 그래프 우측 상단 [⬇] 아이콘 클릭 → PNG 다운로드.
D) 인터페이스 & 알림
  • 툴팁: Function Bar 버튼에 마우스를 올리면 간단 설명이 팝업.
  • 토스트 메시지: 성공, 경고, 오류 상태를 우측 상단 토스트로 즉시 안내.
  • 유효성 오류 배지: 헤더 ‘데이터 유효성 오류’ 버튼에 현재 오류 개수 표시 & 첫 오류로 포커스.
  • 진행률 막대: 대용량 업로드·검증 중 실시간 % 표시.

11. 도움 받기

  • 오류 메시지 화면(F12 → Console 탭)을 캡처해 주시면 문제 해결이 빠릅니다.
