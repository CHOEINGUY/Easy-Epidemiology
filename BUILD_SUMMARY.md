## 🚀 실행 방법

### 방법 1: 파일 직접 열기 (가장 간단)
1. `dist/index.html` 파일을 브라우저에서 직접 열기
2. 모든 기능이 오프라인에서 작동

### 방법 2: 로컬 서버 (권장)
```bash
# dist 폴더로 이동
cd dist

# Python 3
python -m http.server 8080

# 또는 Node.js
npx http-server -p 8080
```

### 방법 3: 자동 실행 스크립트
```bash
# Windows
start-offline.bat

# 또는 테스트 페이지
test-offline.html
```

## 📞 문제 해결

### 일반적인 문제들
1. **Excel 파일 업로드 안됨**
   - 파일 형식 확인 (.xlsx, .xls)
   - 파일 크기 확인
   - 브라우저 콘솔 오류 확인

2. **차트가 표시되지 않음**
   - 데이터 형식 확인
   - JavaScript 콘솔 확인
   - 브라우저 호환성 확인

3. **데이터 저장 안됨**
   - LocalStorage 용량 확인
   - 브라우저 설정 확인
   - 개인정보 보호 모드 해제

### 디버깅 도구
- 브라우저 개발자 도구 (F12)
- 콘솔 로그 확인
- 네트워크 탭에서 리소스 로딩 확인
- Application 탭에서 LocalStorage 확인
