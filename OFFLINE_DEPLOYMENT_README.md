# Easy-Epidemiology Web v1.3 - 오프라인 배포 버전

## 📋 개요

이 버전은 인터넷 연결 없이도 완전히 작동하는 오프라인 배포용 Easy-Epidemiology Web입니다.

## 🚀 실행 방법

### 방법 1: 배치 파일 사용 (권장)
1. `Easy-Epidemiology-Offline.bat` 파일을 더블클릭
2. 자동으로 브라우저가 열리면서 애플리케이션이 실행됩니다

### 방법 2: 수동 실행
1. `dist` 폴더로 이동
2. `index.html` 파일을 더블클릭하거나 브라우저에서 열기

## 📁 배포 파일 구조

```
dist/
├── index.html              # 메인 HTML 파일
├── js/
│   ├── app.js             # 애플리케이션 코드
│   ├── vendors.js         # 외부 라이브러리
│   └── ...                # 기타 JS 파일들
├── fonts/                 # 폰트 파일들
├── favicon.ico           # 아이콘
├── *.xlsx               # 템플릿 파일들
├── *.zip                # 보고서 템플릿들
└── ...                  # 기타 리소스들
```

## ✨ 주요 특징

### 🔒 인증 불필요
- 로그인 없이 바로 사용 가능
- 모든 기능이 즉시 활성화

### 🌐 오프라인 완전 지원
- 인터넷 연결 없이 모든 기능 사용 가능
- 로컬 브라우저 저장소 활용

### 📊 완전한 기능
- 데이터 입력 및 편집
- 통계 분석 (환자-대조군, 코호트 연구)
- 유행곡선 및 잠복기 분석
- 보고서 생성
- 차트 및 그래프 생성

## 🔧 기술 사양

- **프레임워크**: Vue.js 3.2
- **상태 관리**: Vuex 4
- **차트 라이브러리**: Chart.js, ECharts
- **스타일링**: CSS3, Material Icons
- **데이터 처리**: XLSX, PapaParse
- **통계 계산**: JStat

## 📝 사용법

1. **데이터 입력**: DataInput 탭에서 환자 데이터 입력
2. **통계 분석**: CaseControl 또는 CohortStudy 탭에서 분석 실행
3. **유행곡선**: EpidemicCurve 탭에서 차트 생성
4. **보고서 작성**: ReportWriter 탭에서 최종 보고서 작성

## ⚠️ 주의사항

- **브라우저 호환성**: Chrome, Firefox, Edge, Safari 최신 버전 권장
- **데이터 저장**: 모든 데이터는 브라우저 로컬에 저장됩니다
- **파일 크기**: 약 9MB (압축 시 약 1.2MB)

## 🛠️ 문제 해결

### 빌드 오류 시
```bash
npm run build:noauth
```

### 파일이 열리지 않을 때
- 브라우저의 팝업 차단 설정 확인
- 파일 경로에 한글이나 특수문자가 없는지 확인

### 기능이 작동하지 않을 때
- 브라우저 캐시 삭제
- 다른 브라우저로 시도
