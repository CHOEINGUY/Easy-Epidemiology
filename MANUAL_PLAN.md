# 사용설명서 (User Manual) 생성 계획

## 1. 개요

이 문서는 `webpage_office` 프로젝트를 위한 HTML 기반 사용설명서를 제작하는 계획을 정의합니다. 최종 결과물은 **탭(Tab) 인터페이스**를 가진 단일 `user_manual.html` 파일입니다. 이 파일은 별도의 서버 없이 브라우저에서 바로 열 수 있도록 모든 스타일(CSS)과 기능(JavaScript)을 내장합니다.

- **최종 파일명**: `user_manual.html`
- **기술 스택**: HTML, CSS, JavaScript (외부 라이브러리 없음)

---

## 2. 구현 단계

### 1단계: HTML 구조 설계

탭 메뉴와 콘텐츠 영역을 포함하는 기본 HTML 구조를 작성합니다.

- **탭 컨테이너**: 사용자가 클릭할 탭 버튼들을 담는 영역입니다.
- **콘텐츠 컨테이너**: 선택된 탭에 해당하는 설명이 표시될 영역입니다.

```html
<!-- 예시 구조 -->
<body>
  <div class="tab-container">
    <div class="tabs">
      <button class="tab-button active" data-tab="intro">1. 프로그램 소개</button>
      <button class="tab-button" data-tab="execution">2. 프로그램 실행방법</button>
      <!-- (총 11개의 탭 버튼) -->
      <button class="tab-button" data-tab="team">11. 개발진 소개</button>
    </div>
    <div class="tab-content">
      <div id="intro" class="content-panel active">
        <!-- 프로그램 소개 내용 -->
      </div>
      <div id="execution" class="content-panel">
        <!-- 프로그램 실행방법 내용 -->
      </div>
      <!-- (총 11개의 콘텐츠 패널) -->
      <div id="team" class="content-panel">
        <!-- 개발진 소개 내용 -->
      </div>
    </div>
  </div>
</body>
```

### 2단계: CSS 스타일링

사용설명서의 가독성과 사용성을 높이기 위해 깔끔하고 직관적인 디자인을 적용합니다.

- **전체 레이아웃**: 탭과 콘텐츠 영역을 명확히 구분합니다.
- **탭 스타일**:
    - 기본 탭, 활성화된 탭(`active`), 마우스 오버 시(`hover`) 스타일을 다르게 지정합니다.
    - 구글 문서나 최신 웹 앱과 유사한 깔끔한 디자인을 적용합니다.
- **콘텐츠 스타일**:
    - 제목, 본문, 코드 블록, 이미지 등에 적절한 폰트와 여백을 적용하여 가독성을 높입니다.

### 3단계: JavaScript 기능 구현

탭 인터페이스의 동적 기능을 구현합니다.

1.  **이벤트 리스너 설정**: 모든 탭 버튼에 클릭 이벤트를 감지하는 리스너를 추가합니다.
2.  **탭 전환 로직**:
    - 탭 버튼 클릭 시, 모든 탭 버튼과 콘텐츠 패널에서 `active` 클래스를 제거합니다.
    - 클릭된 탭 버튼과 그에 해당하는 ID를 가진 콘텐츠 패널에 `active` 클래스를 추가합니다.
3.  **초기 상태 설정**: 페이지 로드 시 첫 번째 탭('프로그램 소개')이 기본적으로 활성화되도록 설정합니다.

### 4단계: 콘텐츠 작성 (섹션별 내용 초안)

각 탭에 들어갈 내용을 프로젝트 분석 결과에 기반하여 작성합니다.

1.  **프로그램 소개**:
    - 프로그램명: `Easy-Epidemiology Web v1.2` (main.js에서 확인)
    - 목적: 역학 조사를 위한 데이터 입력, 분석, 시각화 및 보고서 생성을 지원하는 웹 기반 도구.
    - 주요 기능: 표 형식 데이터 입력, 환자 특성 분석, 유행 곡선, 임상 증상 분석, 환자-대조군/코호트 연구, 보고서 자동 생성.

2.  **프로그램 실행방법**:
    - **개발 모드**: `npm run serve` 명령어 실행 및 `http://localhost:8080` 접속 방법 안내.
    - **빌드**: `npm run build` 명령어를 통해 정적 파일을 생성하는 방법 안내.
    - **인증 모드**: `.env` 파일 설정에 따른 인증/비인증 모드 차이 설명 (`isAuthRequired` 함수 기반).

3.  **데이터 입력 탭 (`DataInputVirtual.vue`)**:
    - 기능: 대규모 데이터 입력을 위한 가상 스크롤 기반 표.
    - 사용법: 데이터 직접 입력, 복사/붙여넣기, 행/열 추가 및 삭제, 데이터 유효성 검사 기능 설명.
    - Excel 가져오기/내보내기 기능 설명.

4.  **환자특성 탭 (`PatientCharacteristics.vue`)**:
    - 기능: 입력된 데이터를 바탕으로 환자들의 인구학적, 역학적 특성을 요약 및 시각화.
    - 제공 정보: 성별, 연령 분포 등 주요 변수에 대한 통계표와 차트 제공.

5.  **임상증상 탭 (`ClinicalSymptoms.vue`)**:
    - 기능: 환자들의 주요 임상 증상 발현율을 분석하고 시각화.
    - 제공 정보: 증상별 발현율 표, 막대 차트 등.

6.  **유행곡선 탭 (`EpidemicCurve.vue`)**:
    - 기능: 시간에 따른 환자 발생 추이를 보여주는 유행 곡선 차트 생성.
    - 사용법: 증상 발현 시간 간격 설정, 노출 시간 설정 방법 설명.

7.  **환자-대조군 탭 (`CaseControl.vue`)**:
    - 기능: 환자군과 대조군의 특정 요인(섭취 음식 등) 노출률을 비교하여 교차비(Odds Ratio) 계산.
    - 제공 정보: 2x2 표, 교차비(OR), 신뢰구간, p-value 등 통계 분석 결과.

8.  **코호트 연구 탭 (`CohortStudy.vue`)**:
    - 기능: 특정 요인 노출 집단과 비노출 집단의 질병 발생률을 비교하여 상대위험도(Relative Risk) 계산.
    - 제공 정보: 2x2 표, 상대위험도(RR), 발병률 등 통계 분석 결과.

9.  **보고서 작성 탭 (`ReportWriter.vue`)**:
    - 기능: 각 탭에서 분석된 결과를 종합하여 MS Word 형식(.docx)의 보고서 초안 자동 생성.
    - 사용법: 보고서에 포함할 내용 선택 및 생성 버튼 클릭 방법 설명.

10. **사례군 조사 탭 (`CaseSeries.vue`)**:
    - 기능: 특정 사례군에 대한 상세 정보를 목록화하고 요약.
    - 제공 정보: 주요 사례에 대한 핵심 정보 테이블.

11. **기타 프로젝트 개발진 소개**:
    - 이 섹션은 현재 정보를 알 수 없으므로 "개발진 정보를 기입하는 공간입니다."와 같은 플레이스홀더 텍스트로 채웁니다.

### 5단계: 통합 및 최종 파일 생성

1~4단계에서 작성된 HTML, CSS, JavaScript 코드와 콘텐츠를 하나의 `user_manual.html` 파일로 통합하여 최종 결과물을 완성합니다.
